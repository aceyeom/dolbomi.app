## E1 · 육군창업경진대회 (FLAGSHIP)

- **What:** 육군 창업 대회; 국가 *도전! K-스타트업* 파이프라인의 관문. 진짜 엑싯 사례 존재(예: 링티 창업자 2017 수상).
- **Eligibility:** 육군(장교·부사관·병), **2–5인 팀**(개인 불가).
- **Reward:** 본선 수상자 → **포상휴가 2~5일(용사, 부대 내규)** + 상금(대상 ~500만, 총상금 ~2,500만) + 창업멘토링캠프(공가) + 창업지원기관 연계 → 국방 Start-up 챌린지(하반기) → 도전! K-스타트업.
- **Schedule (2026 봄 사이클, 템플릿 — 라이브 사이클 검증; 봄은 5/27 종료):** 서류접수 2.9–4.10 · 서류발표 4.23 · 본선 5.12–15 중 1일(온라인) · 시상 5.27. 육군은 **하반기** 사이클도 운영.
- **Apply:** army-startup.co.kr (예선 폼 + 5분 유튜브 영상 → 본선 발표영상 + 온라인 Q&A).
- **Quest-path:** ① 아이디어 한 줄 정리 ② 시장조사 30분 ③ 팀원 1–4명 모으기 ④ 사업모델 1장 ⑤ 5분 발표영상 촬영 ⑥ 유튜브 업로드 + 접수 ⑦ 본선 Q&A 리허설.
  *(휴가 hook의 가장 깊은 표현: "네 아이디어로 휴가 따자.")*
