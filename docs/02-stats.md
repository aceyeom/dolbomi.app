## 2.1 여섯 개의 스탯 (사회 매핑)

| 인앱 이름 | 현실 역량 | 1차 기회 피드 (→ [04](./04-catalog/README.md)) |
|---|---|---|
| 전투력 **Body** | 체력, 건강 | 체력검정, 사격, 헌혈 |
| 정신력 **Mind** | 회복탄력성, 집중, 독서 | 다독왕, 정신전력 공모전, 회고 |
| 자산력 **Money** | 저축, 금융 리터러시 | 적금, 자기개발비, ISA/청약 |
| 숙련도 **Craft** | 자격증, 취업 스킬 | 국가기술자격, TOEIC, 학점취득, 군 e-러닝 |
| 지휘력 **People** | 리더십, 커뮤니케이션 | 분대장, 멘토링 |
| 담력 **Edge** | 용기, 주도성 | 창업경진대회, 공모전, 회피해온 그것 |

## 2.2 레벨링 수학 (단순하게)

- 각 퀘스트는 `xp` 와 1차 `stat` 을 가진다. **S=10xp, M=25xp, L=60xp**; 기회 서브퀘스트는 보너스 ×1.5 (현실 stakes).
- **스탯 레벨 = floor(sqrt(totalXP / 100))** — 완만한 곡선. 초반 레벨은 빠르게 차고, 후반 레벨은 자랑거리가 된다.
- **담력/Edge XP는 더 희소하고 더 값지다** — 사용자가 회피해온 기회 서브퀘스트에서만 나온다(경로가 7일 이상 방치되면 플래그). 그래서 담력 바가 스크린샷 트로피가 된다.

## 2.3 The Gap (목표 오버레이) — 거의 공짜, 높은 임팩트

온보딩에서 병사가 스탯별 전역 목표치를 설정하고 구체적 목표를 명명한다("정보처리기사 + TOEIC 800 + 적금 만기"). 각 라이브 바 뒤에 **고스트 처리된 두 번째 바**로 렌더. 줄어드는 거리가 *뒤처지는 불안* 을 숫자로 바꾼다.

빌드 비용 ≈ 이미 그려진 바에 스타일링 한 번.
