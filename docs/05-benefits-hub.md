## 5.1 메커닉 — 셀프 필터, 아무것도 저장 안 함

사용자가 필터를 고른다(군별 / 계급 / 관심분야). 자격은 **클라이언트 측에서 계산**한다. 우리는 의도적으로 **수집하지 않는다** — 부모 소득, 카드 데이터, 민감 정보 일체(이게 v3의 함정이었다). 같은 가치, 민감 데이터 저장 0, 신뢰 비용 0.

각 카드 → **"→ 퀘스트로 만들기"** 버튼이 그 혜택을 기회 엔진(→ [03](./03-opportunity-engine.md))에 넘겨 청구 경로(claim-path)를 생성.

## 5.2 혜택 카탈로그 (실제)

| 혜택 | 받는 것 | 청구처 |
|---|---|---|
| 장병내일준비적금 + 매칭 | ~100% 정부 매칭 → 전역 시 ~2,000만 | 은행 앱 + 나라사랑카드 |
| 자기개발비 | 80% 환급, 최대 12만/년(응시료·도서·강좌·용품·문화) | 나라사랑포털 → 자기개발 |
| 국가기술자격 무료검정 | ~82 종목 무료 시험 | 국방부 위탁검정 |
| 대학 원격강좌 | 학기당 6 / 연 12학점, 80% 환급 | 나라사랑포털 → 군 e-러닝 |
| 군 e-러닝 강좌 | 8,000+ 무료/할인(어학·IT·자격증·검정고시) + 전화영어 | 나라사랑포털 |
| 어학 응시료 할인 | 할인된 TOEIC 등 | 나라사랑포털 제휴 |
| 나라사랑카드 혜택 | 할인/포인트 | 카드 혜택 |
| 제대군인·청년 지원 | 취업·정책 (전역 후) | 셀프 필터 → 포털 |

*(각 혜택은 `source_url` 과 검증일을 보유; 허브가 오래된 항목을 플래그.)*
