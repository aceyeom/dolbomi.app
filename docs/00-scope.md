# 00 · 범위 (MVP Scope)

> **상태 박스**
> 우선순위: 전체 빌드의 헌법(framing). 변경 시 모든 파트에 영향.
> 의존성: 없음 (최상위).
> 미해결 질문(Phase 2 후보): FAST-FOLLOW/ DEFERRED 항목의 트리거 기준을 수치로 못박을지.

---

## MVP 한 줄 베팅

**기회 엔진(Part 03)** 이 *지친 병사도 오늘 밤 한 걸음을 떼게* 만들 만큼 좋은 퀘스트 경로를 만들면, 리텐션·바이럴·매출은 따라온다. 그래서 MVP는 거의 전부 **이 한 가지를 탁월하게, 그리고 진짜 카탈로그 위에서** 만드는 데 집중한다.

## IN — 지금 만든다

1. **코어 루프** — 60초 체크인 → 오늘 밤의 3 → 완료 → 바 이동 → 한 줄 (→ [01](./01-core-loop.md))
2. **6스탯 시스템 + 레벨링 + The Gap** (목표 오버레이) (→ [02](./02-stats.md))
3. **전역 D-day + 스트릭** (→ [01](./01-core-loop.md) §1.4)
4. **HUD 잠금화면 위젯**
5. **기회 엔진** — Claude API 기반 상세 진행바 *(센터피스)* (→ [03](./03-opportunity-engine.md))
6. **진짜 기회 카탈로그** (→ [04](./04-catalog/README.md)) — 앱을 "쓸 수 있게" 만드는 실체
7. **혜택 허브** — "군인이라서 받는 모든 것" (셀프 필터, 저장 안 함) (→ [05](./05-benefits-hub.md))
8. **휴가 사다리** — 휴가 집계 (the hook) (→ [03](./03-opportunity-engine.md) §3.4)
9. **리퍼럴 엔진** — 저커밋 퀘스트를 제휴 서비스로 라우팅 (수수료) (→ [06](./06-referral-engine.md))
10. **적금 디스플레이** — 전역 시점 예상 잔액 (시뮬레이터 아님, 단순 표시)
11. **월간 결산** (→ [07](./07-monthly-recap.md))

**AI의 단 하나의 일:** 컨텍스트 → 적응형 퀘스트 경로 (→ [03](./03-opportunity-engine.md) §3.3). MVP에 챗봇·컴패니언 페르소나 없음.
